# Generated ceramic assets

These fictional Terre & Tide product assets were created with the built-in OpenAI image generation workflow on 2026-07-13.

## Files

- `pendant-v2.png` and `pendant-v2.webp`
- `table-lamp-v2.png` and `table-lamp-v2.webp`
- `wall-sconce-v2.png` and `wall-sconce-v2.webp`

The PNG files are the high-resolution transparent deliverables. The WebP files are trimmed and optimized website versions.

## Prompt direction

Each prompt requested one original, opaque, hand-finished ceramic lighting object photographed in a centered three-quarter studio view. The shared palette used terracotta clay, cream mineral slip, deep oxblood glaze, and dark bronze hardware. Text, logos, watermarks, glass, translucency, cast shadows, contact shadows, reflections, room scenes, and extra objects were prohibited.

## Transparency workflow

The built-in generator produced each object on a uniform `#00ff00` chroma-key background. The installed imagegen background-removal helper converted the key color to alpha using a soft matte, despill, and a one-pixel edge contraction. Final alpha and transparent corners were verified before integration.
