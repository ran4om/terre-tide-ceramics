# Terre & Tide

One-page React and Vite concept for a fictional Lisbon ceramic lighting studio. A custom sculptural SVG lamp remains present through four full-height making chapters, changes with scroll progress, and resolves beside the final commission action.

## Run

```bash
npm install
npm run dev
```

Production check: `npm run build`.

## Notes

- The product imagery is an original inline SVG with procedural texture, so no third-party image attribution is required.
- Scroll behavior is native. There is no scroll-jacking.
- Motion follows `prefers-reduced-motion`.
- Contact addresses use the reserved `.example` domain.
